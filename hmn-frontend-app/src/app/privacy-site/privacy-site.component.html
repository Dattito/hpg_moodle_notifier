Datenschutz

So werden deine Daten behandelt
  • Deine Moodle-Anmeldedaten (Benutzername, Passwort) wird niemals gespeichert.
  • Sie werden nur benutzt um einen Token zu erzeugen, mit dem die Arbeitsaufträge aktualisiert werden können. So sieht z.B. ein Token aus: dk87t156mluzsmbw5n81qgkezbgqoc4g
  • Da der Token eine Art Ersatz für das Passwort ist, kann dieser theoretisch dazu verwendet werden, sich auf Moodle in deinem Namen anzumelden. Das wird natürlich niemals passieren, sollte aber gesagt werden.
  • Der Token hat nur eine begrenzte Haltbarkeitsdauer. Wenn dieser abläuft bekommst du eine Nachricht und wirst automatisch abgemeldet und deine Daten werden gelöscht.
  • Deine Telefonnummer & Token werden nur solange gespeichert wie nötig. Das heißt wenn du dich abmeldest wird automatisch alles über dich gelöscht.
